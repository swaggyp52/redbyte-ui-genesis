# Keyboard Shortcuts

Global keyboard shortcuts for RedByte OS.

## Global Shell Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `Ctrl/Cmd+Space` | Open System Search | Anywhere |
| `Ctrl/Cmd+,` | Open Settings | Anywhere |
| `Ctrl/Cmd+Shift+P` | Open Command Palette | Anywhere |
| `Alt+Tab` | Window Switcher | Anywhere |
| `Ctrl/Cmd+W` | Close Focused Window | Anywhere |
| `Ctrl/Cmd+M` | Minimize Focused Window | Anywhere |

## App-Specific Shortcuts

### Logic Playground

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Ctrl/Cmd+Shift+C` | Share Circuit | Copies compressed URL to clipboard |
| `Ctrl/Cmd+S` | Save Circuit | (If implemented) |
| `Ctrl/Cmd+N` | New Circuit | (If implemented) |

### Files App

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Ctrl/Cmd+Shift+N` | New Folder | Opens folder creation modal |
| `F2` | Rename | Renames selected file/folder |
| `Delete` | Delete | Deletes selected file/folder |
